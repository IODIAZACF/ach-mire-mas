Nombre:

class_leyenda

 

Descripción:

 

Clase utilizada para poner una tabla con la leyenda para las teclas a utilizar en la ventana actual a partir de los datos contenidos en un archivo ini o de un array.

 

 

Propiedades:

origen

Si es un array, la clase arma la leyenda utilizando la información del array. Si no, lo toma como el nombre de un archivo .INI.

contenido

Es el HTML resultante para ser mostrado en la ventana.

estilo

Es el estilo CSS que se utilizará para mostrar la leyenda.

 

 

Funciones:

Constructor class_leyenda()

 

 

 

Constructor de la clase class_leyenda:

 

Sintaxis:

 

$query = new class_leyenda();

       

 

armar()

 

 

Función que toma los datos y arma el HTML para guardarlo en (contenido).

 

Sintaxis:

 

$leyenda = new class_leyenda();

$leyenda->origen = "ejemplo.ini";

$leyenda->armar();

echo $leyenda->contenido;

 

 

Dependencias:

Class_clase1, class_clase2, ..

 

Ejemplo:

// Usando un .INI - - - - - - - - -

 

include ("./class/class_leyenda.php");

 

$leyenda = new  class_leyenda();

$leyenda->origen = "EJEMPLO.INI";

$leyenda->estilo = 'leyenda';

$leyenda->armar();

 

echo $leyenda->contenido;

 

// Usando un array  - - - - - - -

 

include ("./class/class_leyenda.php");

$data['LEYENDA1']['TECLA']        = "F1";

$data['LEYENDA1']['ROTULO']       = "AYUDA";

 

$data['LEYENDA2']['TECLA']        = "F2";

$data['LEYENDA2']['ROTULO']       = "OTRA COSA";

 

$leyenda = new  class_leyenda();

$leyenda->origen = $data;

$leyenda->estilo = 'leyenda';

$leyenda->armar();

 

echo $leyenda->contenido;
