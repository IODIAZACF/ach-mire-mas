Nombre:

Class_genera_xml

 

Descripción:

 

Clase utilizada para convertir datos de un array a formato xml.

 

 

Propiedades:

Sql

Es la cadena de la consulta sql que utilizará el xml

Criterio

Es el filtro utilizado

Arbol

Especifica el tag para la tabla en general. Por defecto $arbol = tabla

Data

Es un array que contiene la data leida

contenido

Es el resultado de la generación en formato xml.

Condicion

Especifica los atributos que se deberán establecer para los registros o campos específicos según una condición o filtro.

 

Ej:

$xml->condicion['NOMBRES'][]='registro,color="#FF0000",=,CLEIDYS';

// especifica que pinte todo el registro de color f0000000 cuando el valor del campo

// NOMBRES igual (=) a CLEIDYS

 

$xml->condicion['APELLIDOS'][]='apellidos,color="#00FF00",<=,DA';

// especifica que pinte de la celda de apellidos de color #00FF00 cuando el valor

// del campo APELLIDOS sea igual (=) a DA

 

$xml->condicion['TABLA'][]='tabla,color="#0000FF"';

// especifica que pinte todo el registro de color #0000FF

 

Los operadores condicionales permitidos son los siguientes:

= : para igualdad

>= : mayor o igual que

<= : menor o igual que

<> : distinto de

* : que contenga

 

Funciones:

Constructor class_genera_xml()

 

 

 

Constructor de la clase formulario.

 

Sintaxis:

 

$clase = new class_genera_xml();

 

 

imprime_xml ($data)

 

 

Función que se encarga de guardar

 

Sintaxis:

 

$clase = new class_genera_xml();

:

$clase->imprime_xml();

echo $clase->contenido

 

Parámetros:

 

$data : es un array que contiene la información a desplegar en formato xml.

 

Dependencias:

No tiene dependencias

 

Ejemplo:

include ("./class/class_genera_xml.php");

$data[0][NOMBRES]   = "JUAN ARGENIS";

$data[0][APELLIDOS] = "MONSERRAT ADAM";

$data[1][NOMBRES]   = "JUAN FFFF";

$data[1][APELLIDOS] = "MOSERRAT ADAM";

$data[2][NOMBRES]   = "CLEIDYS";

$data[2][APELLIDOS] = "DE CARREÑO";

$data[3][NOMBRES]   = "MARIA";

$data[3][APELLIDOS] = "DE CARREÑO";

$data[4][NOMBRES]   = "EDSON";

$data[4][APELLIDOS] = "DAZA";

 

 

$xml = new  class_genera_xml();

$xml->data = $data;

$xml->condicion['NOMBRES'][]='registro,color="#FF0000",=,JUAN FFFF';

$xml->condicion['NOMBRES'][]='registro,color="#00FF00",=,CLEIDYS';

$xml->condicion['APELLIDOS'][]='apellidos,color="f0000000",%,DA';

 

$xml->imprime_xml();

echo $xml->contenido;

 

 

Nombre:

Script: genera_xml.php

 

Descripción:

 

Script PHP utilizado para traer el resultado XML para aquellas consultas asíncronas en una página. Específicamente útil para los grids. Los parámetros pueden ser pasados por GET/POST.

 

 

Parámetros:

sql

De uso para guardar el query de se esta generando

criterio

Se utiliza para armar el criterio de la consulta (=, <, >, <=, >=, *)

tabla

Nombre de la tabla o vista en que se hará la consulta

campos

Campos de la tabla (separados por coma) para la consulta

busca

Campos por el cual se realizara la búsqueda

xbusca

Valor de los campos que se realizara la búsqueda

ordcampo

Campos por el cual se realizara el ordenamiento

limite

Cantidad de registro que se pueden devolver

origen

Se especifica cuando los datos a traer provienen de un archivo .ini

operador

Es el ripo de operación a utilizar en el filtrado de la información: los operadores válidos son:

-         = : cuando el filtrado es por una igualdad absoluta

-         < : cuando el filtrado es por valores menores que el especificado

-         > : cuando el filtrado es por valores mayores que el especificado

-         <= : cuando el filtrado es por valores menores o iguales al especificado

-         >= : cuando el filtrado es por valores mayores o iguales al especificado

-         * : cuando el filtrado es por valores que contienen el especificado

-          

Ejemplo:

 

genera_xml.php?tabla=D_EVENTOS&campos=ID_EVENTOS,FACTURAS,FECHA_EVENTO,ID_AFILIADOS&

  busca=ID_AFILIADOS&xbusca=54530&ordcampo=FECHA DESC,FACTURAS&limite=4&operador=>
