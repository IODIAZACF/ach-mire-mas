Nombre:

class_sql

 

Descripción:

 

Clase utilizada para generar los diferentes consultas a la base de datos SQL, bien sean Select’s, Update’s, o Delete’s

 

 

Propiedades:

sql

De uso para guardar el query de se esta generando

criterio

Se utiliza para armar el criterio de la consulta (=, <, >, <=, >=, *)

tabla

Nombre de la tabla o vista en que se hará la consulta

campos

Campos de la tabla (separados por coma) para la consulta

busca

Campos por el cual se realizara la búsqueda

xbusca

Valor de los campos que se realizara la búsqueda. Cuando este campo contiene el valor '*' se buscan todos los registros pero se establece como máximo un valor indicado como constante en la misma clase. Esta constante se llama LIMITE y para los sistemas por defecto se colocó 50.

orden

Campos por el cual se realizara el ordenamiento. Se colocara el tipo de ordenamiento despues del campo.

Limite

Cantidad máxima de registros que se desean retornar

Origen

Se especifica cuando los datos a traer provienen de un archivo .ini

Operador

Es el tipo de operación a utilizar en el filtrado de la información: los operadores válidos son:

-         = : cuando el filtrado es por una igualdad absoluta

-         < : cuando el filtrado es por valores menores que el especificado

-         > : cuando el filtrado es por valores mayores que el especificado

-         <= : cuando el filtrado es por valores menores o iguales al especificado

-         >= : cuando el filtrado es por valores mayores o iguales al especificado

-         * : cuando el filtrado es por valores que contienen el especificado

Regi

Guarda el recorset del Query

Arreglo

Guarda los datos del query ejecutado en un arreglo

Tipo

Tipo de query que se va a ejecutar

my_ini

Arreglo que contiene datos del archivo .INI que fue leído

 

Funciones:

Constructor class_sql()

 

 

 

Constructor de la clase formulario.

 

Sintaxis:

 

$query = new sql();

       

 

obtener_parametros()

 

 

Función que verifica si los datos son pasado por el POS o por el GET

 

Sintaxis:

 

$query = new sql();

$query->obtener_parametros();

       

 

arma_sql($tabla = '', $campos = '', $busca='',$xbusca=’’,$operador=’*’, $orden='', $limite=’’)

 

 

Función que arma el query, es necesario pasarle el tipo de query que se quiere ejecutar antes de llamar esta función.

 

Sintaxis:

$query->arma_sql();

 

Parámetros:

 

$tabla: Nombre de la tabla. Por defecto $tabla = “”.

$campos: Nombre de los campos. Por defecto $campos = “”.

$busca: Nombre de campos separadors por comas para armar el criterio. Por defecto $busca = “”.

$xbusca:  Valores de los campos separadors por comas para armar el criterio. Por defectos $xbusca=“”.

$operador: Tipo de operador logico para filtar la informacion. Por defecto = “*”.

$orden: Nombres separados por coma de los campo a ordenar. Por defecto $ordcampo = “”.

$limite: Numero maximo que se traera la consulta. Por defecto = “”.

 

 

 

ejecuta_query()

 

 

Función que  ejecuta la consulta contra la base de datos.

 

Sintaxis:

$query = new sql();

:

$query->ejecuta_query()

 

 

crea_arreglo()

 

 

Función que crea un arreglo a partir de los datos traidos de una consulta.

 

Sintaxis:

$query = new sql();

:

$query->crea_arreglo()

 

Dependencias:

class_ini, config.

 

Ejemplo:

include ("./class/class_sql.php");

$query = new class_sql();

$query->obtener_datos();

$query->tipo= "select";

$query->arma_sql();

$query->ejecuta_sql();

$query->crea_arreglo();
