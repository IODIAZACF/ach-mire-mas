Nombre:

ini

 

Descripción:

 

Clase utilizada para leer y cargar en un array los valores contenidos en un archivo .INI.

 

 

Propiedades:

parse_constants

Define si serán "parseados" por PHP los distintos valores leidos del .ini

Use_defaults

Define si se usarán valores por defecto en el .ini

 

 

Funciones:

Constructor ini($ini_file = NULL, $default_file = NULL )

 

 

Constructor de la clase ini.

 

Parámetros:

 

$ini_file : Es el nombre del archivo a leer. Por defecto es Null

$default_file : Es el nombre del archivo .Ini que contiene la misma estructura del original pero con valores por defecto. Por defecto es Null.

 

Sintaxis:

 

$ini = new ini("archivo.ini");

       

 

read_ini($filename, $return_ini=false)

 

 

Función que lee el contenido del archivo .ini y opcionalmente retorna una matriz asociada con el contenido del mismo.

 

Sintaxis:

 

$ini = new ini();

$contenido = $ini->read_ini("archivo.ini", true);

$contenido["PERSONAL"]["NOMBRE"]

 

-         $filename: Nombre del archivo .ini a leer.

-         $return_ini: Define si será retornada una matriz asociativa con la data. Por defecto False.

 

La matriz asociada resultante tiene la forma $matriz[<nombre sección>][<nombre campo>]

 

Ejemplo:

 

Archivo.ini

 

[REGISTRO]

NOMBRE=PRUEBA

 

echo $matriz["REGISTRO"]["NOMBRE"]; // retornará "PRUEBA"

 

 

read_defaults($filename)

 

 

Función para leer y retornar los valores por defecto desde un archivo .ini.

 

Sintaxis:

 

$ini = new ini();

$ini->read_defaults("defecto.ini");

 

-         $filename: Nombre del archivo .ini con los valores por defecto.

 

 

Dependencias:

No tiene dependencias

 

Ejemplo:

include ("./class/class_ini.php");

$clase = new ini();

$matriz = $clase ->read_ini("archivo.ini", true);

echo $matriz["VENTANA"]["ANCHO"];
