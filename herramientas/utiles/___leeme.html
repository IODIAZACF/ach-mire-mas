Nombre:

path.php -> path($root = true)

 

Descripción:

 

Contenido en utiles/path.php. Función para retornar la ruta en http del script que se ejecuta actualmente ($root = _path), la ruta de la primera carpeta contenedora  ($root = _root) o la ruta del servidor ($root = _root).

 

 

 

path($root = _root)

 

 

Sintaxis:

 

$path = path(); // retorna la ruta del servidor

$path = path(_path); // retorna la ruta del script

       

Dependencias:

No tiene dependecias

 

Ejemplo:

include ("path.php");

 

echo path(_root).'<br>';

echo path(_folder).'<br>';

echo path(_path);
